MOtoNMS: Matlab MOtion data elaboration TOolbox for NeuroMusculoSkeletal applications

2013-present • Matlab


MOtoNMS logo

MOtoNMS is a Matlab toolbox able to read motion data stored in C3D files and process markers trajectories, ground reaction forces, and EMG signals for the OpenSim and CEINMS software tools.



Details, User Manual, Examples, and Public Forum are available at this page.
Code and instructions on how to use it are hosted on GitHub together with the online documentation.
Developed with: Alice Mantoan


Multidimensional Cubic B-Spline

2011 -present• C + +


MCBS

This project presents a C + + implementation of multidimensional cubic B-spline for the fast estimation of lengths and three-dimensional moment arms for musculotendon actuators. A MATLAB wrapper is also included in the package.




Details can be found at this page.
Code and instructions on how to use it are hosted on GitHub.
Developed with: Massimo Sartori
Additional contributor: Elena Ceseracciu
